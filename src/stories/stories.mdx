
## Storybook for seedling UI components


This project utilizes Storybook v9 for its component documentation. The UI component library is built upon [Shadcn](https://ui.shadcn.com/).    

- In the top toolbar, you'll find a light/dark theme switch and a multi-language switch.
- The Github repository: [xiongjia/seedling ](https://github.com/xiongjia/seedling)
